# Changelog

All notable changes to DocNav will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-01-15

### Added
- PyPI package publication
- GitHub repository setup
- Comprehensive documentation
- Professional CLI and Python API
- Multi-format document support (PDF, DOCX, TXT, Excel, PowerPoint, CSV)
- Multiple LLM provider support (OpenAI, Gemini, Claude)
- Source citations in answers
- Smart document chunking
- Vector search capabilities

### Changed
- Updated author information
- Fixed Unicode path handling for Windows
- Improved error handling

### Fixed
- CLI argument parsing issues
- Document processing bugs
- LLM integration problems

## [1.0.0] - 2026-01-14

### Added
- Initial release of DocNav
- Core document processing functionality
- Basic CLI interface
- Python API
- Embedding model support
- LLM integration framework
